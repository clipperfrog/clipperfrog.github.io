Abstract
Web 3.0 is the new internet. It runs on each phone and computer as a node. We host a small
amount of the internet on each device. We call each node lumber.
The node runs the following hardware:
- A storage medium
- A graphics card
- A network switch
For owning a node you get paid an amount each hour, optimistically more than minimum wage.
Each node will host a free wi hotspot, allowing for VPNs of a mass scale.
What have we learnt from Web 2.0, and how can we use that knowledge to generate a new
awless next generation?
We have achieved communication over a shared internet allowing users not in the same
physical location to communicate over almost any distance. Since 1876, technology has been
improving in astronomical leaps, and although Moore's law has since plateaued, we have built
Quantum Computing.
What is Quantum Cryptography and what threat does it pose to current encryption?
1. PGP is an example of an asymmetric encryption system - ref that is commonly used on
the secure and anonymous darknet TOR market sites.
2. PGP uses the RSA cryptosystem to deliver the session key; it simply encrypts the
randomly-generated session key with Bob's public key and then appends the
RSA-encrypted session key to the beginning of Alice's session-key-encrypted document.
The document and session key are then sent together to Bob. - ref
Key Points:
1. PGP is RSA/DSA based
2. PGP runs on darknet markets where criminals and buyers believe they are secure.
3. When quantum computers become robust enough to run Shors algorithm on a large
scale, the most used asymmetric algorithms, utilized for digital signatures and message
encryption, such as RSA, (EC)DSA, and (EC)DH, will no longer be secure.
4. John Graham-Cumming, chief technology ocer of internet infrastructure company
Cloudare, said there's a lot of uncertainty: It could take ve years before quantum
computers can crack encryption or it could take 20. - CNET

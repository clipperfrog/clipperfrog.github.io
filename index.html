NVMe Storage - Cold Storage.
-
NVMe is one of the fastest storage mediums however DDR4 (volatile
memory ), allows read/write speeds upwards of 45gb/s compared to
NVMe’s measly 2000MB.
Volatile high-speed memory.
-
If we were to use a low response time volatile memory solution such as
DDR4, it can technically replace Cold Storage. As the network can never
be shut down without (Size of Nodes * (volatile memory + non-volatile
memory))/Size of DB going below 1, the network can scale without the
chance of deletion.
